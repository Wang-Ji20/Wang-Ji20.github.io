emacs

Posted at 2023-09-06 Comments CS  text toolbox 

今天开始学习使用 emacs.

指令

C-x C-c 退出 emacs.

C-g 取消当前操作.

C-x C-f 打开文件.

C-x C-s 保存文件.

C-x C-w 另存为.

C-space 设置标记.

C-w: 剪切

M-w: 复制

C-y: 粘贴

C-k: 剪切到行尾

C-x u 撤销

C-x 2 分割窗口, 用 C-x o 切换窗口

C-s C-r 搜索
