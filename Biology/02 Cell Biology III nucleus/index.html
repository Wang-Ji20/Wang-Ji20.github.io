    核孔复合体

    ![[Pasted image 20220311152116.png]] ![[Pasted image 20220311152137.png]]

    核孔80～120nm，核孔复合体120～150nm。30多种多肽，1000多个蛋白质分子组成，分子量1.25亿？？如果不是蛋白质围成，很快就会因为流动闭合。 核孔复合体的功能：细胞核与细胞质物质交换的双向亲水通道 核孔复合体的功能直径约9nm，离子，小分子可以扩散通过，而RNA，蛋白等需要通过信号识别与主动运输过程，消耗ATP。 亲核蛋白质（karyophilic protein）：需要进入细胞核发挥功能的蛋白，含有核定位信号。 核定位信号(nuclear localization signal, NLS)：引导蛋白质穿过核孔进入细胞核的氨基酸特定序列，主要由4-8个带正电氨基酸组成，可位于多肽序列的任何部分。

    细胞间期 -- 染色质 有丝分裂的时候 -- 染色体，缠绕紧密

    中期染色体的结构

    着丝粒（centromere)：连接两条姐妹染色单体的部位，又称主缢痕（primary constriction)。将染色单体分为：短臂（p，petite）和长臂（q，queue） ![[Pasted image 20220311152612.png]]

    端粒（telomere):染色体两个端部特化结构，由物种特异性的串联重复DNA组成。端粒的功能是维持染色体的完整性和独立性。 ![[Pasted image 20220311153042.png]] ![[Pasted image 20220311153048.png]]端粒也是一段高度重复的序列。重要功能：维持染色体的稳定性。否则末端会被降解，也会有粘性，粘到别的染色体身上。 ![[Pasted image 20220311153422.png]]

    次缢痕（secondary constriction): 染色体上除主缢痕外其它的浅染缢缩部位，不同染色体次缢痕的数目，位置，大小不同，可作为鉴定染色体的标记。

    核仁组织区（nucleolar organizing region, NOR) 位于次缢痕，是rDNA（含rRNA序列）所在部位，与间期细胞核仁形成有关。人类有五对，所以最多十个核仁。

    随体（satellite):位于染色体末端的球形染色体节段，通过次缢痕与染色体主体部分相连。

    染色体的三种基本功能元件

    自主复制序列（autonomously replicating sequence, ARS)：确保染色体能够自我复制。

    着丝粒(centromere)：保证染色体能平均分配到子细胞中。

    端粒（telomere)：保持染色体的独立性和稳定性

    染色体数目

    ![[Pasted image 20220311153904.png]] ![[Pasted image 20220311153909.png]] 红毛猩猩在加里曼丹岛，印度尼西亚。黑猩猩大猩猩都在非洲，很久之后才接触到。所以红毛猩猩只叫猩猩，别的都叫xx猩猩。

    人类没有24对染色体其实是一件叫人惊讶的事。大猩猩有24对染色体。黑猩猩也是。在猿类动物里我们是个例外。在显微镜下面，我们与其他猿类动物最大、最明显的区别就是我们比它们少一对染色体。原因很快就弄清楚了，并不是猩猩的染色体到我们这儿丢了一对，而是在我们的身体里，两对猩猩的染色体融合在了一起。人类染色休学第二大的一条(2号染·色体)是两条中等大小的猩猩染色体融合起来形成的。这从人类染色体与相应的猩猩染色体上那些暗带的排列就可以看出来。
    　教皇让·保罗二世于1996年10月22日在天主教科学院的讲话中提出，古猿与现代人类之间存在一个“本体的断裂”——这个断裂点就是上帝向动物的一个分支注入了人的灵魂的时刻。这种说法可以使教廷与进化论达到和解。也许这个本体的飞跃是发生在两条猩猩染色体融合的时候吧，也许编码灵魂的基因就在人类2号染色体的中间的地方?
