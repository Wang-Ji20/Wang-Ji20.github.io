## 核孔复合体
![[Pasted image 20220311152116.png]]
![[Pasted image 20220311152137.png]]

核孔80～120nm，核孔复合体120～150nm。30多种多肽，1000多个蛋白质分子组成，分子量1.25亿？？如果不是蛋白质围成，很快就会因为流动闭合。
核孔复合体的功能：细胞核与细胞质物质交换的双向亲水通道
核孔复合体的功能直径约9nm，离子，小分子可以扩散通过，而RNA，蛋白等需要通过信号识别与主动运输过程，消耗ATP。
亲核蛋白质（karyophilic protein）：需要进入细胞核发挥功能的蛋白，含有核定位信号。
核定位信号(nuclear localization signal, NLS)：引导蛋白质穿过核孔进入细胞核的氨基酸特定序列，主要由4-8个带正电氨基酸组成，可位于多肽序列的任何部分。

## 染色质和染色体
细胞间期 -- 染色质
有丝分裂的时候 -- 染色体，缠绕紧密

### 中期染色体的结构

着丝粒（centromere)：连接两条姐妹染色单体的部位，又称主缢痕（primary constriction)。将染色单体分为：短臂（p，petite）和长臂（q，queue）
![[Pasted image 20220311152612.png]]

端粒（telomere):染色体两个端部特化结构，由物种特异性的串联重复DNA组成。端粒的功能是维持染色体的完整性和独立性。
![[Pasted image 20220311153042.png]]
![[Pasted image 20220311153048.png]]端粒也是一段高度重复的序列。重要功能：维持染色体的稳定性。否则末端会被降解，也会有粘性，粘到别的染色体身上。
![[Pasted image 20220311153422.png]]
   

次缢痕（secondary constriction): 染色体上除主缢痕外其它的浅染缢缩部位，不同染色体次缢痕的数目，位置，大小不同，可作为鉴定染色体的标记。

核仁组织区（nucleolar organizing region, NOR) 位于次缢痕，是rDNA（含rRNA序列）所在部位，与间期细胞核仁形成有关。人类有五对，所以最多十个核仁。

随体（satellite):位于染色体末端的球形染色体节段，通过次缢痕与染色体主体部分相连。

### 染色体的三种基本功能元件

自主复制序列（autonomously replicating sequence, ARS)：确保染色体能够自我复制。

着丝粒(centromere)：保证染色体能平均分配到子细胞中。

端粒（telomere)：保持染色体的独立性和稳定性
